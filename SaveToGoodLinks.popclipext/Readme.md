# Save to GoodLinks - PopClip Extension

URL을 선택하면 GoodLinks 앱에 자동으로 저장하는 PopClip 확장 기능입니다.

## 기능

- 선택한 텍스트가 URL인지 자동 감지
- URL을 GoodLinks에 빠르게 저장 (UI 없이 바로 저장)
- HTTP, HTTPS URL 지원

## 설치 방법

1. `SaveToGoodLinks.popclipext` 폴더를 더블클릭하여 설치
2. PopClip이 자동으로 확장 기능을 설치합니다

## 사용 방법

1. 웹 페이지나 문서에서 URL을 선택 (드래그)
2. PopClip 메뉴에서 "GoodLinks" 아이콘 클릭
3. 선택한 URL이 GoodLinks에 자동 저장됩니다

## 요구사항

- macOS에 GoodLinks 앱이 설치되어 있어야 합니다
- PopClip 2023.9 이상

## 예제

다음과 같은 URL을 선택하면 작동합니다:
- `https://example.com`
- `http://example.com`
- `https://www.apple.com/iphone`

## 문제 해결

확장 기능이 작동하지 않는다면:
1. GoodLinks 앱이 설치되어 있는지 확인
2. PopClip을 재시작
3. URL만 선택했는지 확인 (주변 텍스트 없이)
